# Always-On Robot Learning from Demonstrations

Bachelor thesis: Multi-task learning for Franka Emika robot using behavioral cloning.

## Dependencies
- Python 3.11
- SIC Framework: `pip install social-interaction-cloud`
- panda-py
- Redis

## Project Structure
- `src/learning/` - BC and context-aware BC implementations
- `src/robot_control/` - Robot interface code
- `demos/` - Demo scripts
- `data/` - Collected demonstrations
- `models/` - Trained models
